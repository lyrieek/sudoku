# Lyrieek-Sudoku

天天数独

## Getting Started

好玩的数字游戏，在传统的模式上加入了新玩法，使用Flutter编写，多平台可用

## Features

- 经典模式，破解50个数独谜题
- 找茬模式，挑出数独中的所有的错误位，充满运气与挑战
- 纠错模式，数独中出现了3对位置错误的块，把它们纠正过来
